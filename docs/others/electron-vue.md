## 安装 electron-vue 脚手架并新建工程

1. `npm install -g vue-cli`

2. 新建工程 `vue init simulatedgreg/electron-vue esp-desk`

   ![Fomh1x.png](https://s2.ax1x.com/2019/01/03/Fomh1x.png)

3. 安装依赖 `npm install`

4. 运行程序 `npm run dev`
